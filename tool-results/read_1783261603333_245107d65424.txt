   671‚Üí   671‚ÜíKey decisions:
   672‚Üí   672‚Üí- Equity definition: totalEquity = costBasis + unrealizedPnl. This is
   673‚Üí   673‚Üí  the standard "what you put in + what the market gave you back" view.
   674‚Üí   674‚Üí  Cost basis always counts (even for positions whose live quote fails);
   675‚Üí   675‚Üí  unrealizedPnl skips nulls (treated as 0). Documented inline.
   676‚Üí   676‚Üí- totalExposure only counts mark-to-market-able positions (skip null
   677‚Üí   677‚Üí  marketValue). This means a portfolio with a large failed-quote
   678‚Üí   678‚Üí  position will show a low exposurePct ‚Äî honest, and the frontend can
   679‚Üí   679‚Üí  cross-reference with /portfolio to show "Source unavailable" per row.
   680‚Üí   680‚Üí- perPosition.marketValue is number (not nullable) per the RiskSummary
   681‚Üí   681‚Üí  type in src/lib/types.ts. When a quote fails, marketValue=0,
   682‚Üí   682‚Üí  weightPct=0, pnlPct=0. The /portfolio GET endpoint carries the
   683‚Üí   683‚Üí  explicit null + error per position; the frontend should use that for
   684‚Üí   684‚Üí  per-row error display and use /risk/summary for the aggregated view.
   685‚Üí   685‚Üí- VaR requires ‚â•2 positions with return series (per spec example). With
   686‚Üí   686‚Üí  1 position only, VaR is null even if its own series is well-formed ‚Äî
   687‚Üí   687‚Üí  this is a conservative choice the frontend should know about. Could
   688‚Üí   688‚Üí  be relaxed later if single-position VaR is desired.
   689‚Üí   689‚Üí- Correlation matrix only includes pairs where BOTH instruments have a
   690‚Üí   690‚Üí  return series (‚â•5 log-returns). Self-pairs (a==a) explicitly value=1
   691‚Üí   691‚Üí  (trivial, no helper call). Cross-pairs are time-aligned via
   692‚Üí   692‚Üí  two-pointer intersection before being passed to `correlation()`.
   693‚Üí   693‚Üí- Drawdown reconstructed from costBasis √ó cumprod(1+r) using log returns
   694‚Üí   694‚Üí  for the portfolio aggregate. Daily log returns are small (<0.05), so
   695‚Üí   695‚Üí  1+r ‚âà exp(r) and the reconstructed equity curve shape is correct.
